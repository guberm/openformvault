# v1 product scope

## In scope from v1

- Accounts and trusted devices.
- Encrypted local vault on every client.
- Zero-knowledge sync.
- Logins.
- Identities.
- Safenotes.
- Bookmarks.
- Folders/pinned/recent/search.
- TOTP attached to logins.
- Password/passphrase generator.
- RoboForm/CSV import with dry-run preview.
- Manual item create/edit/delete.
- Startup screen selection per device.
- Lock/auto-lock and local biometric/PIN unlock where platform supports it.
- Chrome website fill/save/update.
- Android AutofillService fill/save/update.
- Windows desktop vault/editor/sync/settings UI.
- Sync status, errors, conflicts, retry state.

## Deferred after foundation

- Shared folders with permissions.
- Emergency access.
- Enterprise admin/SCIM/SSO/policies.
- Passkey storage/sync until platform constraints are fully designed.
- Windows app-login automation beyond explicit user-triggered prototype.
